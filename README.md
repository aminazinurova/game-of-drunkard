# Карточная игра в «пьяницу»

Индивидуальный учебный проект №2 по дисциплине **«Языки программирования»**.

## Описание проекта
Данная программа представляет собой консольный пошаговый симулятор популярной карточной игры в «пьяницу» для двух игроков.

## Технические особенности реализации
* **Собственная структура данных:** В проекте принципиально не используются стандартные списки Python (`list`) для ведения игры. Колоды игроков реализованы вручную на базе **динамической очереди (FIFO)** и связных узлов памяти.
* **Высокое быстродействие:** Благодаря хранению прямых указателей на начало и конец списка, операции извлечения и добавления карт выполняются за константное время **$O(1)$**, что обеспечивает мгновенный расчет лимита в $10^6$ ходов.
* **Модульная архитектура:** Программа разделена на изолированное вычислительное ядро (`logic.py`) и интерфейсный модуль представления (`main.py`) с циклическим интерактивным меню.
* **Отказоустойчивость («Защита от дурака»):** Реализована многоуровневая валидация данных с перехватом исключений `ValueError` и `FileNotFoundError`.

## Правила симуляции
1. В игре участвует 10 уникальных карт (номиналы от `0` до `9`). Каждый игрок получает на старте по 5 карт.
2. Старшинство карт определяется их номиналом, за исключением **«правила нуля»**: младшая карта `0` побеждает самую старшую карту `9`.
3. Выигранные в раунде карты отправляются под низ колоды победителя строго в фиксированном порядке: сначала карта первого игрока, затем карта второго игрока.
4. При зацикливании колод и достижении лимита в 1 000 000 ходов симуляция останавливается и выдает результат `botva`.

## Структура репозитория
* `logic.py` — модуль логики (описание классов узла, очереди и игрового цикла).
* `main.py` — интерфейсный модуль (консольное меню, валидация ввода, чтение/запись файлов).
